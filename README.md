# 📊 Smart Data Chatbot (LLM + SQL)

An AI-powered chatbot that allows users to upload CSV/Excel files and ask questions in natural language.

---

## 🚀 Features
- Upload CSV/Excel files
- Natural Language → SQL Query
- Clean table output
- Chat interface

---

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- Ollama (Llama3)
- SQLite

---

## ▶️ Run Project

pip install -r requirements.txt  
streamlit run app.py  

---

## 🧠 Example Queries
- Show employees in Sales
- Count employees by department
- Show employees from Manchester
